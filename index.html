Nuestros Cursos

Básico

Este curso está diseñado para principiantes que desean dominar las herramientas esenciales de Excel. En cuatro semanas, aprenderás a navegar por la interfaz de Excel, a gestionar datos, a aplicar formatos básicos y a crear gráficos simples. Ideal para quienes buscan comenzar con confianza en Excel.

  • Navegación y gestión básica de datos: Introducción a la interfaz de Excel, manejo de hojas de cálculo, celdas, y rangos.
  • Formateo de celdas: Aplicar formatos como negrita, cursiva, colores de fondo y bordes.
  • Fórmulas básicas: Aprender a usar fórmulas como SUM, AVERAGE, MIN, y MAX para operaciones simples.
  • Creación de gráficos: Cómo crear y personalizar gráficos de columnas, líneas, y circulares para presentar datos de forma visual.
  • Manejo de datos simples: Cómo ordenar y filtrar datos para análisis básico.

Al finalizar, estarás equipado con las habilidades fundamentales necesarias para trabajar con hojas de cálculo.

Intermedio

El curso intermedio de Excel es perfecto para quienes ya tienen conocimientos básicos y desean ampliar sus habilidades. Durante esta semana, aprenderás a:

  • Manipulación de texto y fechas: Uso de funciones como CONCATENATE, LEFT, RIGHT, DATE, y WEEKDAY para manipular y analizar datos textuales y de fechas.
  • Funciones de búsqueda y referencia: Implementación de funciones como VLOOKUP, HLOOKUP, INDEX, y MATCH para extraer información específica de grandes conjuntos de datos.
  • Análisis de datos avanzado: Uso de herramientas como filtros avanzados, validación de datos, y tablas de datos para un análisis más profundo.
  • Tablas dinámicas: Creación y personalización de tablas dinámicas para resumir y analizar grandes cantidades de datos de manera rápida y eficaz.
  • Formateo condicional: Aplicación de reglas de formateo condicional para resaltar datos importantes automáticamente.

Este curso te ayudará a gestionar y analizar datos de manera más efectiva.

Avanzado

Este curso avanzado está diseñado para usuarios experimentados que desean llevar sus habilidades de Excel al máximo nivel. Durante cinco semanas, aprenderás a integrar Excel con otras fuentes de datos, a utilizar funciones financieras y estadísticas avanzadas, y a proteger tus hojas de cálculo con técnicas de seguridad avanzadas.

  • Integración y conexión de datos: Cómo importar y exportar datos, y conectar Excel con fuentes de datos externas como bases de datos SQL o servicios web.
  • Automatización de tareas: Utilización de funciones avanzadas como IF, SUMIF, COUNTIF, y SUBTOTAL para automatizar cálculos complejos.
  • Funciones financieras y estadísticas: Aplicación de funciones como IRR, NPV, STDEV, y PERCENTILE para realizar análisis financieros y estadísticos.
  • Protección y seguridad de datos: Técnicas para proteger hojas de cálculo, como el uso de contraseñas, proteger celdas específicas, y asegurar la integridad de los datos.

Este curso está diseñado para usuarios que buscan dominar Excel a un nivel profesional.
