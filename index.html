Nuestros Cursos

Básico

Este curso básico de Excel está diseñado para aquellos que son nuevos en el uso de Excel. A lo largo de una semana, aprenderás a:

  • Navegar por la interfaz de Excel, crear y guardar hojas de cálculo.
  • Formatear celdas para mejorar la presentación de datos.
  • Utilizar fórmulas simples como SUM y AVERAGE.

Al finalizar, estarás equipado con las habilidades fundamentales necesarias para trabajar con hojas de cálculo.

Intermedio

El curso intermedio de Excel es perfecto para quienes ya tienen conocimientos básicos y desean ampliar sus habilidades. Durante esta semana, aprenderás a:

  • Utilizar funciones condicionales como IF, COUNTIF y SUMIF.
  • Crear y modificar tablas dinámicas para el análisis de datos.
  • Explorar la potente función VLOOKUP para buscar datos en tus hojas de cálculo.

Este curso te ayudará a gestionar y analizar datos de manera más efectiva.

Avanzado

En el curso avanzado de Excel, profundizarás en técnicas avanzadas de análisis de datos. Durante una semana, descubrirás cómo:

  • Utilizar funciones avanzadas como SUMPRODUCT y OFFSET.
  • Realizar análisis de escenarios para evaluar diferentes resultados.
  • Optimizar problemas y proteger tus hojas de cálculo.

Este curso está diseñado para usuarios que buscan dominar Excel a un nivel profesional.
